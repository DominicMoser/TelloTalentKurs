# P03_Dance
## Ziele
- Durch den Raum bewegen
- Tricks ausführen
- Code mit Schleifen wiederholen.
- Kennenlernen von Codeblöcken, welche mit `:` beginnen und alle dazugehörigen befehle mit 4 Leerzeigen eingerückt werden.

## Neue Befehle
- `for i in Range():`
- `while <Value> == True:`
- `flip_back()`
- `flip_forward()`
- `flip_right()`
- `flip_left()`

## Mögliche Probleme
- Falsches einrücken der Zeilen.
- Vergessen von `:`
